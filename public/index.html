✅ **If the link does not work for you, copy this code instead and run it locally:**

<!DOCTYPE html> <html> <head> <title>Chat App</title> </head> <body> <h2>Chat App (Offline Version)</h2> <div id="messages"></div> <input id="input" placeholder="Type here..."> <button onclick="send()">Send</button> <script> function send() { const msg = document.getElementById("input").value; const box = document.getElementById("messages"); const p = document.createElement("p"); p.textContent = msg; box.appendChild(p); document.getElementById("input").value = ""; } </script> </body> </html>

✅ Anyone can copy this offline version into a file called index.html and open it in any browser.
